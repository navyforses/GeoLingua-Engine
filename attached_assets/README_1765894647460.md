# ­ЪїЇ GeoLingua

**P2P рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃЌрЃљ рЃЏрЃљрЃарЃЎрЃћрЃбрЃцрЃџрЃћрЃўрЃАрЃў Рђћ Bolt/Uber рЃАрЃбрЃўрЃџрЃўрЃА**

рЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃБрЃфрЃ«рЃЮрЃћрЃЌрЃерЃў рЃЊрЃљ рЃБрЃфрЃ«рЃЮрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃерЃў.

---

## ­Ъј» рЃарЃљ рЃљрЃарЃўрЃА GeoLingua?

GeoLingua рЃљрЃарЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ рЃАрЃљрЃЊрЃљрЃф:
- **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА** рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃЮрЃА рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃў рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃћрЃюрЃљрЃќрЃћ
- **рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃА** рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃЏрЃБрЃерЃљрЃЮрЃА
- **AI рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃў** рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃљ 24/7 рЃЊрЃљрЃЉрЃљрЃџ рЃцрЃљрЃАрЃљрЃЊ

### рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў:
- ­Ъћ┤ **рЃљрЃ«рЃџрЃљрЃЋрЃћ рЃЏрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ** Рђћ Broadcast рЃАрЃбрЃўрЃџрЃў, рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃўрЃюрЃф рЃљрЃўрЃдрЃћрЃЉрЃА
- ­ЪЊЁ **рЃЊрЃљрЃњрЃћрЃњрЃЏрЃўрЃџрЃў** Рђћ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃЊрЃљрЃ»рЃљрЃЋрЃерЃюрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЊрЃарЃЮрЃА
- ­Ъцќ **AI рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃў** Рђћ рЃўрЃљрЃцрЃў рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃАрЃљрЃБрЃЉрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- ­Ъњ░ **рЃЌрЃћрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃцрЃљрЃАрЃў** Рђћ рЃќрЃЮрЃњрЃљрЃЊрЃў, рЃАрЃљрЃЏрЃћрЃЊрЃўрЃфрЃўрЃюрЃЮ, рЃўрЃБрЃарЃўрЃЊрЃўрЃБрЃџрЃў

## ­ЪЏа№ИЈ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃАрЃбрЃћрЃЎрЃў

### Backend
- **Node.js + Express** Рђћ REST API
- **Socket.io** Рђћ рЃарЃћрЃљрЃџ-рЃбрЃљрЃўрЃЏ рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ
- **Supabase** Рђћ PostgreSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
- **Twilio** Рђћ рЃЋрЃўрЃЊрЃћрЃЮрЃќрЃљрЃарЃћрЃЉрЃў
- **Stripe** Рђћ рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃћрЃЉрЃў
- **OpenAI** Рђћ AI рЃЌрЃљрЃарЃ»рЃўрЃЏрЃљрЃюрЃў (Whisper + GPT-4 + TTS)

### Frontend (рЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў)
- **React Native** Рђћ iOS + Android
- **Expo** Рђћ рЃАрЃгрЃарЃљрЃцрЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ
